Creating AWS S3 bucket and deploying a website via bucket

I started by creating an S3 bucket and uploading my project files. At first, it was just storage, but after configuring access properly, I was able to load the site directly from the S3 object URL in the browser. Seeing my project live from cloud storage was a big step.

STEPS
- Created and set-up S3 bucket with best practice methods
- Set-up policy to make the bucket non vulnerable and 
- Configures access to the bucket properly
- Uploaded website files (i.e. HTML, CSS, etc.) files directly into the bucket
- Copied URL and Successfully loaded the website homepage through browser.


What I focused on while doing this:

- Understanding how S3 actually serves files vs just storing them
- Working with permissions to make objects publicly accessible
- Getting more comfortable with AWS networking and infrastructure
- Connecting the dots between different AWS services instead of treating them in isolation

Learning, building, and refining step by step.
